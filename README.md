## Описание
Этот репозиторий создан для выполнения лабораторной работы №3 по системе контроля версий Git.  
Цель работы — освоить работу с коммитами, ветками, merge и rebase, а также управлять удалённым репозиторием на GitHub.

---

## Основные действия, выполненные в ходе лабораторной

1. Модификация файла `requirements.txt`:
   - Добавлены строки `"зачет"`, `"не зачет"`, `'зачет (pr_feature)'`, `"зачет (после отката)"` в разных коммитах.
2. Решение конфликтов:
   - Конфликты при rebase и merge решались вручную с сохранением всех корректных изменений.
3. Использование Git команд:
   - `git add`, `git commit -m`, `git log`, `git switch`, `git merge`, `git rebase`.
4. Работа с удалённым репозиторием:
   - Настройка GitHub и репозитория, push изменений.
5. Сохранение истории коммитов:
   - `history.txt` — полный журнал всех изменений в репозитории.
   - `diff.txt` — разница между коммитами.
   
---

## Как работать с репозиторием

1. Клонируйте репозиторий:
   git clone https://github.com/spyspooky11/lab3_git.git
   
2. Проверьте финальное состояние файла requirements.txt:
    cat requirements.txt

3. Проверьте состояние файла history.txt:
    cat history.txt
